### 环境变量

```bash
# Redis 连接配置
REDIS_HOST=localhost          # Redis 主机
REDIS_PORT=6379               # Redis 端口
REDIS_PASSWORD=your_password   # Redis 密码(可选)
REDIS_DB=0                     # 数据库编号
REDIS_POOL_SIZE=20             # 连接池大小

# 熔断器配置
CACHE_CIRCUIT_THRESHOLD=5     # 失败阈值(连续失败N次后熔断)
CACHE_CIRCUIT_TIMEOUT=60       # 熔断超时(秒),超时后尝试恢复
```

### TTL 配置

| 数据类型 | TTL | 抖动 | 说明 |
|---------|-----|------|------|
| 会话数据 | 3600s (1小时) | ±10% | 对话历史缓存 |
| 槽位数据 | 1800s (30分钟) | ±10% | 意图槽位缓存 |
| 用户偏好 | 604800s (7天) | ±10% | 长期偏好存储 |

### 启动 Redis

```bash
# Docker Compose (推荐)
docker-compose up redis

# 或 Docker
docker run -d -p 6379:6379 redis:7-alpine

# 验证
redis-cli ping  # 应返回 PONG
```
