### 2.2 核心组件职责

| 组件 | 文件 | 职责 |
|------|------|------|
| `QueryEngine` | `query_engine.py` | 总控中心，编排8步工作流程 |
| `IntentClassifier` | `intent/classifier.py` | 三层意图分类（缓存→关键词→LLM） |
| `SlotExtractor` | `intent/slot_extractor.py` | 槽位提取（目的地、日期、预算等） |
| `LLMClient` | `llm/client.py` | LLM客户端封装，支持流式输出 |
| `ToolRegistry` | `tools/registry.py` | 工具注册表 |
| `ToolExecutor` | `tools/executor.py` | 工具并行执行器 |
| `PromptBuilder` | `prompts/builder.py` | 提示词构建器 |
| `ContextGuard` | `context/guard.py` | 上下文守卫（前置/后置处理） |
| `MemoryService` | `memory/` | 记忆服务（RAG检索、持久化） |
| `SubAgentOrchestrator` | `subagent/orchestrator.py` | 多Agent编排器 |
